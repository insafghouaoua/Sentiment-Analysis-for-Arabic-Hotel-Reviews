Project Overview

This project focuses on analyzing Arabic hotel reviews and automatically classifying them into sentiment categories (Positive / Negative). Built as a complete end-to-end AI pipeline — from raw data to model evaluation — using the HARD (Hotel Arabic-Reviews Dataset).

Problem Statement

Online hotel platforms contain vast volumes of Arabic reviews. Manual analysis is inefficient, and Arabic's morphological richness makes automated processing difficult. Valuable insights remain unused due to the scarcity of effective Arabic NLP tools.

Our Solution

We developed an intelligent pipeline that normalizes Arabic text, removes diacritics and dialects, generates rich embeddings, and trains multiple ML/DL models — achieving up to 94% accuracy with fine-tuned CAMeL-BERT using QLoRA.

Prompt Engineering, Embeddings + classifiers (RNN, CNN, LSTM), and Fine-tuning transformers.
